## 1、firstcpp
这个文件夹里面包含了一些基础的cpp指令包括cout和查看变量的虚拟地址和内存大小的指令

## 2、testvar
这个文件夹里面包含了变量和字面量的定义与运算

## 3、vararea
这个文件夹里面介绍了作用域与生命周期

## 4、testconst
这个文件夹里面介绍了编译时常量和运行时常量

## 5、testauto
这个文件夹介绍了定义类型auto的使用

## 6、iflogic
这个文件里面介绍了if逻辑判断的使用，以及算术运算符，逻辑运算符等使用方法

## 7、test_string
这个文件夹里面介绍了string的用法，以及字符串的特性，截断，空串，字符串到数字stoX转换，数字到字符串to_string转换，字符串的拼接、查找、替换等


## 8、test_enum
这个文件夹里面介绍了枚举类型enum的使用
